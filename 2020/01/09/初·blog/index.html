启动服务器

在hexo文件夹下右键选择 git bash here，进入后输入hexo s启动服务器。之后在浏览器进入http://localhost:4000/ 进入属于你自己的blog，你的私人小窝。

加文件后的操作

当你在sources文件夹里加入文件后，需要在git操作窗口输入 hexo g -d 更新在你的git仓库里的文件才可以继续使用你加入文件后的blog。

更改主题后

当你更改主题后，在git控制台输入 hexo clean 更新老主题的缓存，以避免当你进入blog之后还留在之前那个主题一脸懵逼。
